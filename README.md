# Local_LLM101

A practical guide to building, running, and optimizing local LLM systems—from single-GPU setups to multi-GPU AI servers.

Local_LLM101 is a systems-focused handbook for understanding how modern Large Language Models (LLMs) run on local hardware. The goal is to bridge the gap between using local AI tools and understanding the infrastructure, hardware, and software that power them.

Whether you're running a 7B model on a consumer GPU or planning a multi-GPU inference server, this repository aims to provide the practical knowledge needed to make informed decisions.

## What You'll Learn

* How LLMs work under the hood
* GPU memory math and VRAM requirements
* Why memory bandwidth matters
* Building AI machines from 1 GPU to 16 GPUs
* Choosing the right inference engine
* Running local RAG systems
* Giving local models access to the web
* Performance optimization techniques
* Hardware selection and deployment strategies

## Who Is This For?

* AI enthusiasts running local models
* Developers building AI applications
* Engineers interested in inference systems
* Homelab enthusiasts
* Anyone curious about the infrastructure behind local AI

## Philosophy

This repository focuses on practical understanding.

Rather than repeating documentation, the goal is to explain *why things work the way they do*, provide real-world examples, and help readers make better decisions when building local AI systems.
